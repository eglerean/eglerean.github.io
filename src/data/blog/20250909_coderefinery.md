Kick-off of another CodeRefinery workshop, for which I am one of the main organizers. This year we used an extended format running from September 9 to October 22, with more exercises and more teachers.

**Topics covered**:
- Introduction to version control with Git
- Collaborative distributed version control
- Reproducible research
- Social coding and open software
- Documentation
- Jupyter notebooks
- Automated testing
- Modular code development

**Instructors included**: Johan Hellsvik, Jarno Rantaharju, Sabry Razick, Radovan Bast, Michele Mesiti, Gregor Decristoforo, Ina Pöhner, Oskar Taubert, Bjørn Lindi, Andrew Mashchak, Thomas Pfau, Yu Tian, and others.

Recordings are available on YouTube and materials remain accessible at the [workshop page](https://coderefinery.github.io/2025-09-09-workshop).
